## Descripción del proyecto
En este proyecto se realiza un montaje básico con Arduino para hacer que un LED encienda y apague cada segundo.

Se escogió este ejemplo porque es una práctica sencilla para comenzar a trabajar con la estructura de un repositorio en Git, organizar los archivos del proyecto y documentar lo esencial de manera clara.

## Diagrama simple

Arduino UNO  
└── Pin 13 → Resistencia → LED → GND
